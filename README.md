% NRG paper figures

This repository contains scripts to recreate the figures from the NRG paper.

### Required R libraries

```{r}
library(SingleCellExperiment)
library(scater)
library(statmod)
library(pheatmap)
library(TSCAN)
library(destiny)

library(cowplot)
library(ggplot2)
library(ggthemes)
library(ggbeeswarm)
library(GGally)
library(RColorBrewer)
library(network)
library(sna)
library(intergraph)
```
